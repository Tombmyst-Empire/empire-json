Welcome to empire-json's documentation!
==========================================

This module contains utility functions to operate and transform JSON-like dictionaries.
Although it is mainly focused on JSON-like dictionaries, the functions behavior *may* work
on any type of dictionaries, but it is not guaranteed.

**Precision**: A single JSON object in this project, and throughout Empire's other projects
are refer to as *records*.
