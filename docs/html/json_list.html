JSON List¶

class ejson.json_list.JSONList¶

Utilities for list of JSONs (or dictionaries)

static map_by_field(field_name: str, json_list: empire_commons.types_.JsonList, ignore_when_field_is_missing: bool = False, remove_field_from_mapped_json: bool = False) → empire_commons.types_.JsonType¶

From the provided json_list, this method creates a JSON object that refers every list’s records by the specified field_name value in these items.

Parameters:

  • field_name – The field name to use to create the result.

  • json_list – The list of JSON objects

  • ignore_when_field_is_missing – When True, ignores any list’s record that does not contain field_name: it does not add this record to the result.

  • remove_field_from_mapped_json – When True, field_name is deleted from the resulting JSON object

Returns:

A JSON object
